HTML Elements

Core tags you should memorize. Use <code> when showing literal tags.

Common HTML Tags
Tag Purpose Example
<h1>...</h1> Top-level heading (use once per page ideally) <h1>Welcome</h1>
<p>…</p> Paragraph of text. <p>Hello world.</p>
<a href="#">…</a> Link to another page or section. <a href="/about">About</a>
<img src="..." alt="..."> Image element. <img src="logo.png" alt="Company Logo">
<ul>…</ul> Unordered list. <ul><li>Item 1</li><li>Item 2</li></ul>
<ol>…</ol> Ordered list. <ol><li>Item 1</li><li>Item 2</li></ol>
<li>…</li> List item (used within <ul> or <ol>). <li>First item</li>
<div>…</div> Generic container for block-level content. <div class="container">Content</div>
<span>…</span> Generic container for inline content. <span class="highlight">Text</span>
<form>…</form> Form for user input. <form action="/submit" method="post"> … </form>
<header>…</header> Header section. <header><h1>Title</h1>…</header>
<section>…</section> Section of content. <section><h2>Title</h2>…</section>
<footer>…</footer> Footer section. <footer>© 2025 Company</footer>
<nav>…</nav> Navigation section. <nav><ul><li>Home</li><li>About</li></ul></nav>
<meta> Metadata about the HTML document. <meta charset="UTF-8">
<link> Link to external resources like CSS files. <link rel="stylesheet" href="styles.css">
<title>…</title> Document title. (on browser tab) <title>My Page</title>
<figure>…</figure> Container for figures and illustrations. <figure><img src="image.jpg" alt="Description">
<figcaption>…</figcaption> Caption for a figure. <figcaption>This is a caption.</figcaption>

CSS Selectors

Common CSS selectors to target HTML elements.

Selector Basics
Selector Matches Example
element All elements of that type p { color: blue; }
.class All elements with that class .highlight { background: yellow; }
#id Element with that specific ID #header { font-size: 2em; }
element.class Elements of that type with that class div.container { padding: 10px; }
element, element Multiple specified elements h1, h2 { margin-bottom: 0.5em; }
parent > child Direct child elements ul > li { list-style: none; }
ancestor descendant Descendant elements div p { line-height: 1.5; }
element + sibling Adjacent sibling elements h1 + p { margin-top: 0; }
element[attr="value"] Elements with specific attribute value a[target="_blank"] { color: red; }
element:hover Styles applied on hover button:hover { background: blue; }
element::pseudo-element Specific part of an element p::first-line { font-weight: bold; }
* All elements * { box-sizing: border-box; }

Box Model

Understanding the CSS Box Model is crucial for layout and design.

Box Model Properties
Property What it Controls Example
width Sets the width of the content area. width: 200px;
height Sets the height of the content area. height: 100px;
padding Space between content and border. padding: 10px;
border Border around the padding and content. border: 2px solid black;
margin Space outside the border. margin: 15px;
box-sizing Defines how width and height are calculated. box-sizing: border-box;
overflow Controls content overflow behavior. overflow: hidden;
outline Outline around an element. outline: 1px solid red;
